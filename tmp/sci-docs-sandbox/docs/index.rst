dummy-gp documentation
======================

``dummy_gp`` is a small Gaussian process sandbox package used to test formal
documentation workflows. The package deliberately exposes a narrow public API so
the docs can focus on three things:

* how the kernel behaves scientifically,
* how users should apply the helper utilities, and
* whether the Sphinx configuration cleanly renders reference material.
